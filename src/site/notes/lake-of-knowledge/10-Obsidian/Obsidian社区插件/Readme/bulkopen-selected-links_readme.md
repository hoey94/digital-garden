## Readme(翻译）

下面是 [[bulkopen-selected-links\|bulkopen-selected-links]] 插件的自述翻译

批量打开选定的链接

该插件允许用户在编辑模式下轻松打开所有选定的链接，简化了同时编辑多个链接的过程。使用该插件，用户不再需要手动逐个链接打开编辑模式，节省时间并提高效率。只需选择链接，让插件完成剩下的工作。

功能

在编辑模式下打开所有选定的链接
