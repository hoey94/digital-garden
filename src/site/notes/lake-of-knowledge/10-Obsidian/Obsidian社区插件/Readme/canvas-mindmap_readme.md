# Obsidian 画布思维导图

- 按下 `Enter` 创建同级卡片；
- 按下 `Tab` 创建子级卡片；
- 删除卡片会自动重新排列布局；
- 按下 `Alt` + `←/→/↓/↑` 在卡片之间导航；
- 按下 `Cmd / Ctrl` + `←/→/↓/↑` 创建浮动卡片；
