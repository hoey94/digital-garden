# 更新前置元数据的修改日期

当您编辑一个笔记时，此插件将自动更新前置元数据/ YAML 中的修改属性。

与其他插件不同的是，它不使用文件系统的修改时间。只有当您在 Obsidian 中实际工作时，前置元数据才会更新。

## 选项

- 指定您喜欢的任何 YAML 字段名称。
- 使用 [MomentJS 格式](https://momentjs.com/docs/#/displaying/format/) 指定日期格式。
- 排除您不希望自动更新的文件夹。这对于存储脚本或 Templater 模板的任何位置都很重要。
