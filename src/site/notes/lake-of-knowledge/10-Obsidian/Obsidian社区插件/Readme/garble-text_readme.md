# 文本混淆

此插件在 Obsidian 中公开了 `garbleText()` 函数。使用命令面板（`ctrl+p`）查找函数**切换文本混淆**。

在 Obsidian 中混淆文本会将整个应用程序中的所有内容（笔记、侧边栏等）转换为不可读的文本（并模糊图像），以便您可以在不暴露敏感数据的情况下进行截屏。

悬停时，混淆的文本将恢复为原始文本。
