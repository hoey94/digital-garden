# Obsidian Excel 插件

Obsidian-Excel 插件将 [x-spreadsheet](https://github.com/myliang/x-spreadsheet)（一种功能强大的电子表格工具）集成到 Obsidian 中。您可以在您的保险库中存储和编辑 `xlsx` 文件。
