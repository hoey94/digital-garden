## Copy as HTML Obsidian 插件

这个简单的插件使用 showdown 将选定的 markdown 转换为 HTML，并将其复制到剪贴板。

命令名称：`Copy as HTML: Copy as HTML command`

像 Slack 和 Google Docs 这样的平台允许用户将 HTML 粘贴到文本区域，并保持原始格式的笔记。

当您需要复制笔记以与他人共享工作时，该插件非常有用。
