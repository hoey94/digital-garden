使用方法

通过以下方式，可以切换项目并将**已完成的待办事项**移动到单独的已完成区域：

- 点击左侧边栏上的图标，或者
- 使用快捷键：`Control + Enter` *(默认)*

偏好

### 设置标题级别

您可以使用从 `1` 到 `6` 的数字设置已完成区域的级别，分别代表 `h1` 到 `h6`。

默认为 `2`，即 `h2`。

### 设置标题名称

默认为 `已完成`。

显示图标或不显示

如果您不喜欢侧边栏图标，可以将其关闭并使用命令代替。
