# Obsidian 自动添加 Front Matter

自动在 Front Matter 中添加 `title`、`id`、`datetimeCreate`、`datetimeUpdate`，然后重新排序它们。它不会影响 Front Matter 中的其他键。

使用方法

当文件发生更改时，此插件会更新前置元数据。
