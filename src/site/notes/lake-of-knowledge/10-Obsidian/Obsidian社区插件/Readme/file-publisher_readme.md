# 创建一个新文件进行发布

1. 创建一个包含以下前置信息的模板

   ```
   ---
   title:
   excerpt: ''
   timestamp: {{date:MMM D, YYYY}}
   ---
   ```

2. 创建一个新的笔记。
3. 通过执行 `Templates: Insert template` 命令插入模板。
4. 编写你的笔记。
5. 右键点击笔记，选择 `Publish file` 进行发布。
