## Readme(翻译）

下面是 [[last-modified-timestamp-in-status-bar\|last-modified-timestamp-in-status-bar]] 插件的自述翻译

在状态栏中显示当前笔记的最后修改时间戳！

## 用法

时间戳会频繁更新（每 2 秒一次，并且在切换笔记时更新）。

![演示](./img/demo.png)

## 可定制性

- 启用/禁用切换按钮，用于**创建**和**最后修改**时间戳
- 时间戳格式，用于**创建**和**最后修改**时间戳
- 状态栏中的时间戳标题，用于**创建**和**最后修改**时间戳
- **最后修改**时间戳的刷新间隔
