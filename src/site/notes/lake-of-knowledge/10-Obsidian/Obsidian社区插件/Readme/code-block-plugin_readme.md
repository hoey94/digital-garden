# 代码块插件

该插件将选定的文本或粘贴的文本转换为带有自动编程语言检测的代码块。可以在插件设置中选择可检测的语言。语言检测由 highlight.js 提供，虽然不是完美的，但大部分情况下都能正常工作。如果猜错了，从列表中移除该语言，然后再试一次。
