# 关于 Obsidian

- [Obsidian](https://obsidian.md/) 是一个基于 Markdown 文件的笔记应用
- 它是可扩展的
关于这个插件

- 按下 CTRL + P（Mac 上为 cmd + P），然后运行“重新排序列表中的复选框”
- 简单的插件，将选中的复选框放在列表的末尾
- 为了方便起见，使用键盘快捷键绑定该命令

输入：

- [x] 购物
- [ ] 清洁
- [x] 阅读
- [ ] 淋浴

输出：

- [ ] 清洁
- [ ] 淋浴
- [x] 购物
- [x] 阅读
