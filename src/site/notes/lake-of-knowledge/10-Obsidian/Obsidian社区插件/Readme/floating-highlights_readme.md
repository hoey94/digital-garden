## Obsidian Float

- 这是一个为 Obsidian 设计的插件，通过为阅读模式下的滚动添加更好的动画效果，增强了 Obsidian 当前的高亮功能，使得更加关注高亮内容。
- 仅在阅读模式下工作

`注意`：在高亮一个单词时，包含高亮单词的整个文本块也会被动画化。
