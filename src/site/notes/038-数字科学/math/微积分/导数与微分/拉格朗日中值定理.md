设函数 $f(x)$ 满足: 
1. 在闭区间 $[a,b]$ 连续
2. 在开区间 $(a,b)$ 可导
3. $f(a)=f(b)$
则至少存在一个点 $c$, 使得：
$$f ^ { \prime } ( c ) = \frac { f ( b ) - f ( a ) } { b - a }$$
其中 $c$ 满足 $a \lt c \lt b$
