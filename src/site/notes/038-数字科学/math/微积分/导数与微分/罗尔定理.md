设函数 $f(x)$ 满足: 
1. 在闭区间 $[a,b]$ 连续
2. 在开区间 $(a,b)$ 可导
3. $f(a)=f(b)$
则 $(a,b)$ 至少存在一个点 $c$, 使得：
$$f ^ { \prime } ( c ) = 0$$
证明过程使用了费马引理, 此处略
罗尔定理可以推广为[[038-数字科学/math/微积分/导数与微分/拉格朗日中值定理\|拉格朗日中值定理]]
