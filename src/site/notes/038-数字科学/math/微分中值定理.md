微分中值定理是微积分中的一个重要定理，它表明在某些条件下，函数在某个区间内的平均变化率等于在该区间内某一点的瞬时变化率。

**微分中值定理**：

设函数f(x) 在闭区间 $[a,b]$ 上连续，在开区间 $(a,b)$ 内可导。则存在一个点 $c$，使得：
$$f ^ { \prime } ( c ) = \frac { f ( b ) - f ( a ) } { b - a }$$

其中 $c$ 满足 $a \lt c \lt b$

微分中值定理表明在某个区间内，函数在某一点的导数等于函数在该区间两端点连线的斜率。在这个区间内，总存在一点的瞬时变化率等于平均变化率。
